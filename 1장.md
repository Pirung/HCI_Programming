# 1장
--------------------------
### 멀티미디어 탄생
    - 90년대 이후 ICT 환경이 발전 -> 디지털 형태로 변환
    - 누구나 멀티미디어 제작 가능
    - 효과적인 소통 수단, 콘텐츠의 질이 상승
    
### 멀티미디어 개념
    - 다수(Multum) + 정보 유형(medium)
    - 디지털화한 복합 구성 매체
    - 디지털 미디어를 2개 이상 결합해 목적을 가진 콘텐츠 생성
    - ICT 환경을 기반으로 시스템과 상호작용해 지식 정보 제공하는 기술

    - 사운드, 비디오: 연속적, 현실 세계에서 수집
    - 애니메이션: 연속적, 컴퓨터에 의한 합성
    - 이미지: 이산적, 현실 세계에서 수집
    - 텍스트, 그래픽: 컴퓨터에의한 합성, 이산적

### 멀티미디어 특징
    - 현실과 유사한 환경을 콘텐츠
    - 정보의 비선형적 탐색: 사용자 우선, 낮은 비용, 시간 공간 제약 초월
    - 사용자 간 상호작용
    - 대용량 멀티미디어 데이터 처리와 검색
      - 실시간 처리, 복제성(무한 저장 복제 변환 가능), 데이터 저장, 빠르게 정보 검색
    - 콘텐츠 유통 방식의 변화
      - 시공간 초월해 원하는 정보 검색, 가공, 저장, 공유 가능  
    
    표현 미디어 -> 저장 미디어 -> 전송 미디어 
    -> 제시 미디어 -> 지각 미디어

### 전통 미디어
    - 정보통신 기술이 도입되기 이전부터 사용된 통신 수단
    - 레거시 미디어: 전통적인 언론, TV, 신문, 잡지, 라디오
    - 인쇄 전파 비디오 뉴미디어 시대로 발전,
    - ICT 기술을 기반으로 한 미디어 시장의 진화와 변화, 현장감, 속도감이 부족해 하락

### 뉴미디어
    - 전통 미디어와는 다른 전송 방식과 콘텐츠를 사용하는 제품이나 서비스
    - 전통 미디어 영향력 감소, 인터넷 포털 매체가 각광
    - 전통 미디어를 복합적으로 활용, 단순한 정보 전달뿐 아니라 콘텐츠 생성 기능까지 포함
    - 2010년대 이후 본격화, 미디어의 순환 구조에 전체적인 변화와 기능의 근본적 변화
    - SNS, 상호작용 뉴스

### 뉴미디어 특성
    - 정보의 디지털화: 
      - 컴퓨터 이용으로 인해 모든 정보 디지털 신호
    - 정보의 통합화: 
      - 아날로그 형태의 매체가 비트로 통합
      - 정보가 유기적으로 융합되어 사용자에게 전달
    - 정보의 영상화: 
      - 정보가 영상으로 표현 다양한 디바이스 화면을 통해 전달
    - 정보의 상호작용성: 
      - 쌍방향 전송 가능, 
      - 원하는 정보만 선택, 
      - 정보 제공자와 사용자간 상호작용을 통해 새로운 정보 생성
    - 정보의 비동기화: 
      - 시간 공간 제한 극복
    - HCI 기술과 멀티미디어의 융합: 
      - 인간 컴퓨터 상호작용 기술이 멀티미디어 환경에 적용해 인간이 컴퓨터를 더 유용하게 사용

### 스마트 미디어
    - 사용자와 상호작용 가능, 시공간 제약 없이 융복합 콘텐츠 제공 매체
    - 쌍방향 미디어, 상호작용성을 통해 타인과 자유롭게 소통 가능
    - 스마트폰 등장 후 시공간 제약 극복

### 멀티미디어 주변 환경의 변화
    - 컴퓨터의 성능 향상과 정보 통신 기술 발전
      - 대용량 미디어 저장장치 일반화, 초고속 정보통신망 보급
      - 멀티미디어가 급속도로 발전
    - 사용자 인터페이스의 발전
      - 초기에는 문자로 입력, 그래픽 기반 환경에서 마우스 사용, 최근은 멀티터치, 사람의 동작 음성 인식
      - UI: 인터페이스 기능에 대한 것만 정의
      - UX: 인터페이스에 대한 사용자의 상호작용과 반응까지
    - 디지털 기술의 발전
      - 대용량 멀티 미디어 데이터 압축 복원 기술 발전
        - 저장 공간 절약, 정보 전송 효율성 향상
      - 디지털 디바이스 개발, 무선 네트워크 환경 구축, 스마트폰 대중화
    - 전통 미디어 간의 융합
      - IPTV, 스마트 TV
    - 다양한 멀티미디어 저작 도구의 개발 -> 멀티미디어 발전의 가속화

### 컴퓨터 성능 향상
    - 프로세서의 처리 능력
      - 병렬 분산 처리 방식, GPU 활용해 성능 향상
    - 주기억장치
      - 컴퓨터 실행시 프로그램, 데이터를 기억
      - 주기억장츠를 위한 기억 매체로 반도체 기억장치 사용
      - 성능은 기억 용량과 동작 속도로 평가
    - 대용량 저장장치
      - 1980년대 이전까지 보조 기억장치로 플로피 디스크, 하드디스크 사용
      - CD-ROM, DVD, 블루레이 개발로 멀티미디어의 발전이 급격히 진행
      - SSD는 낸드 플래시 적용해 HDD보다 성능면에서 탁월 고성능 고용량

### 사용자 인터페이스의 발전
    - 사용자 인터페이스는 사용자와 컴퓨터 간 정보 소통 구간
    - 시스템에서 가장 먼저 접하는 부분
    - 사용자와 시스템 간 원활하게 의사소통 하게 해주는 물리, 가상적 매개체
    - 입력 출력 원리로 작동
      - 사용자가 시스템을 조작하면 시스템이 결과 표시
    - UI는 대부분 GUI, 최근 모바일, 스마트 TV 등 발전에 핵심적인 기술
    - CUI -> GUI -> NUI 순
      - NUI: 사람의 동작과 음성 등을 활용

## 디지털 기술의 발전
    - 아날로그: 연속적이 신호 또는 물리량 형태로 정보를 표현하는 방식
    - 디지털: 0, 1 비트 단위의 이진수 이용해서 정보를 이산적인 수치로 표현하는 방식
    - 압축 복원 기술의 발전
      - 대용량 멀티미디어 데이터를 다른 형식으로 변환 -> 용량 감소
      - 압축률: 멀티미디어 정보 용량의 감소
      - 멀티미디어 데이터는 정보 형태에 따라 각각 다른 압축 알고리즘 사용
      - EX) 이미지 JPEG 동영상 MPEG
    - 디지털 디바이스의 발전
      - 아날로그 매체가 디지털화 되서 기능이 변경 혹은 확장 제품 출시
        - 스마트폰, 스마트 TV 등
    - 정보통신 기술의 변화
      - 고속 무선 네트워크 환경
        - 3G LTE 5G 순으로 상용화
      - 클라우드 컴퓨팅
        - 가상공간에 위치한 서버에 여러명이 사용하게 저장 공간 제공
      - 초연결 사회의 구현
        - 사물과 사물이 인터넷과 모바일로 연결되 통신이 가능한 유비쿼터스 컴퓨팅 현실화
      - 빅데이터의 분석과 활용
        - 방대한 빅데이터를 전세계 어디서나 즉시 전송, 분석 가능
        - 다양한 마케팅에 활용
    - 인간 생활의 변화
      - 디지털 미디어의 보편화, 대중화로 디지털에 긍정적, 활용, 적응 수준 향상
      - 스마트폰과 디지털 매체에 대한 의존도 급증
      - 인터넷을 통한 보안 위협과 사생활 침해에 단점 존재
      - 불평등한 정보 접근으로 디지털 격차 발생 가능성
