### 디지털 전환 시대
    - 최근 ICT 환경에서 나타나는 변화: 정보화로 표현하기 불가능
    - 현시대는 빅데이터를 기반으로 컴퓨터가 스스로 학습, 알아서 탐색
    - EX)자율주행, 뉴럴, 머신러닝, 인공지능, 딥러닝 등
    - 4차산업혁명: 디지털 혁명이 우리의 삶에 접목되어 산업 구조 변화

### 디지털 개념의 특징
    - 통합성
      - 다양한 아날로그 미디어가 0,1 비트 기반 통합
      - 통합성에 의해 멀티미디어 탄생
    - 복제성
      - 비트를 복제 -> 추가적인 비용 부담 X
      - 정보의 손실을 변형,손상 없이 무한정 저장, 복제, 변환 가능
    - 전파성
      - 유통의 효율화 -> 추가 비용 없이 무한대로 복제 및 전파 가능
    - 저장성
      - 하드디스크, 메모리 등의 디바이스에 저장
      - 디스플레이, 프린터 등의 인터페이스를 통해 표현
      - 수정 추가 삭제 등의 변형 가능
    - 연결성
      - 디지털의 통합성, 복제성, 전파성, 저장성-> 연결성으로 수렴
    - 비경합성
      - 디지털의 경우 다수가 동시에 서비스 이용 가능
    - 공유성
      - 사용자가 많이자면 추가비용 없이 가치가 오름
      - 디지털은 인터넷 환경에서 수월한 공유, 연결 유통을 극대화해 표준화된 플랫폼
    - 유동성
      - 인터넷 연결, 단절, 재연결의 용이성 -> 결합과 분리 쉬워짐
    - 지능화
      - 인공지능 로봇

### 디지털 기술과 소비자의 변화
    - 개인의 역량 강화
      - 전문성 대중화, 지식 생산 참여
      - 크라우드 소싱: 기업의 생산과 서비스 과정에 대중 참여
      - 집단 지성: 다수가 서로 협동하거나 경쟁하여 쌓은 지적, 능력의 결과로 얻은 집단적 능력
    - 개인의 영향력 행사: 인플루언서 마케팅
    - 소셜 러닝: 학습에 SNS를 결합한 교육 방식

    - Z세대
      - 95~05년에 출생한 사람
      - 인터넷이 보편화된 환경에서 태어나 스마트폰 접함
      - 동영상 콘텐츠 소비 비중 높음
      - 모든 정보의 공개, 공유와 자유로운 검색을 보장하는 투명성 신봉
      - SNS 소통, 유행에 민감

### 디지털 시대의 세대별 소비 패턴
    - 베이비붐 세대
      - 우리나라의 경제 성장, IMF 경제 위기 경험 -> 소비보다 저축 중시
    - 밀레니엄 세대
      - 워라밸 중요, 가심비, 나홀로 소비, 개인 자유 중시, 욜로족
    - 파이 세대
      - 개성, 자기계발투자, 경험에 중점 소비, 욜로족 유사, 소유보다 합리적 공유 지향, 즉흥적 소비
    - 뉴트로
      - 현대적인 감성으로 복고 문화를 새롭게 즐기는 문화
      - 10~20대가 경험하지 못한 옛것에 신선함을 느껴 재해석하고 소비

### 디지털 시대의 마케팅
    - 스타벅스 마케팅
      - 블록체인 기술로 원두 이력 관리
      - 사물 인터넷과 클라우드로 연결해 실시간 레시피 공유
      - 소비자의 취향을 분석, 인공지능으로 특정 시간대에 메뉴 추천
    - 바이럴 마케팅
      - 콘텐츠에 기업, 제품을 자연스럽게 삽입, 간접광고, 공유 전달
    - 인플루언서 마케팅
      - 자신의 브랜드를 단 제품을 직접 제작, 판매
      - 명확한 광고 표시 없이 제품 서비스 홍보로 인해 기만 행위 가능성

### 스마트 미디어와 소통 방식의 변화
    - SNS와 스마트폰 결합된 새로운 소통 방식
      - 인터넷을 통한 개방적 소통의 효율성을 배경
      - 일부에 나타나는 비도덕성이 사이버 공론장의 정당성을 파괴
    - 새로운 소통방식
      - 소통의 접근성 강화, 정보 공유 공간 확장, 언제 어디서나 상호작용 가능
      - 단점: 소외된 사람들에게 스트레스, 집단, 패거리 의식의 확산, 보안 문제

### 스마트 미디어 이용 형태의 변화
    - 1단계: 기술 지배 시기
      - 모바일 기기 기술 자체 대한 신선함 작용, 기술 홍보를 위해 컨텐츠 다량 생산
    - 2단계: 사용자 지배 시기
      - 모바일 기기가 제공하는 기술 이용, 기술보다는 서비스 콘텐츠 관심
      - 취향과 요구에 부응하는 콘텐츠 생산, 사용성에 대한 평가
    - 3단계ㅣ 사용자가 지배당하는 시기
      - 모바일 기기 중심, 종속되서 심리적 몰입 상태에 빠지는 시기, 콘텐츠 지속 제공, 현재 상황

### 스마트 미디어 환경의 순기능
    - 빠른 정보 수집: 실시간 소통
    - 영향력과 여론 형성
    - 개인의 콘텐츠 생산 및 배포
    - 개인의 사회의 융합 촉진
      - 정서적, 사교적 소통에 사용
      - SNS는 공통의 관심사 중심으로 거대한 공동체를 형성하는 데 큰 역할

### 스마트 미디어 환경의 역기능
    - 정보의 신뢰 하락, 개인 정보 노출, 정보 격차 발생
    - 사이버 명예훼손과 모욕, 관심병 환자 급증, 청소년 스마트폰 과의존

### 스마트폰 증후군
    - 노모포비아: 스마트폰 없으면 심리적 공포, 불안감
    - 과의존 속성:
      - 현저성: 스마트폰이 일상에서 제일 우선, 조절 실패 등

### 디지털 인터페이스
    - 초기에는 DVI로 대용량의 데이터 전달 가능
    - DVI는 PC 관련 업체들이 개발
    - HDMI는 DVI의 각종 케이블을 연결하는데 불편함을 해결하기 위해 탄생
    - 영상과 음향 동시에 전송, 표준 인터페이스

### USB 프로토콜과 썬더볼트 프로토콜
    - USB프로토콜

      - 모든걸 합침
      - USB보다 빠름

### 사용자 인터페이스의 진화
    - 사용자 인터페이스의 개념
      - 사용자가 가장 먼저 접하는 화면의 전체적인 구성과 작동 방식
      - 시스템과 사용자가 상호 작용하는 것
    - 사용자 인터페이스 설계 규칙
      - 사용자가 필요한 요소를 쉽게 찾고, 원하는 작업 지원
      - 배우기 쉽고 기억하기 쉬워야 함
      - 시각적 소통을 통해 정보 획득과 작업 수행을 도와야 함
      - 사용자가 의도한 결과를 명확하게 얻어야 함
      - CUI GUI NUI 개념 암기

### 사용자 경험의 개념
    - 특정 제품이나 서비스를 직간접적으로 이용하면서 느끼는 총체적인 경험과 감정
    - 2000년대 후반부터 UI 보다 UX 용어 더 많이 사용
    - UX 설계를 위해서는 제품이나 서비스 개발 모든 단계에서 사용자 조사 진행 필요

### 인터랙션과 HCI
    - 인터랙션의 개념
      - 사람과 컴퓨터 사이의 동적인 상호작용
    - 상호작용성
      - 인터낵션 정도 단위, 최근 사용자 간 소통 효율성 중점, 집단지성을 통해 생산 주도
    - 인터렉션 디자인
      - 사람과 객체간에 다양한 형태로 소통이 일어나게 하는 것이 목적
      - 초기에는 컴퓨터 환경, 최근은 디자인 개념 적용해 관계 경험에 대한 개념 확대
      - 최근에는 다중 감각 인터랙션이 발전

    - HCI 개념
      - 사람들이 쉽고 편한하게 시스템과 상호작용 연구
      - 초기 컴퓨팅 시대의 중심은 기술 먼저 개발, 인간은 적응
      - 인간 중심으로 변경되어 집단 디지털 콘텐츠 등 관계 중요
      - HCI가 중요한 이유
        - 디바이스들이 전문가가 아닌 일반 사용자가 사용하는 도구로 바뀜
        - 사용자에게 편리성 향상과 만족감을 주어야 함, 비지니스 적으로 향상된 성능
        - 유용하게 사용할 수 있도록 설계
      - 최종 목표는 사용자에게 최적의 경험을 제공
        - 조건: 유용성, 사용성, 감성
      - 일상생활의 보편적인 도구로 확대
      - 소셜 인터랙션
      - 영역의 경계 파괴
