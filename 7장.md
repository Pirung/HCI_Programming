### 애니메이션
    - 개념: 정적인 매체에 역동성, 사운드 등 추가해 생명을 불어넣는 기법 또는 영화
    - 전통적인 애니메이션
      - 일반적으로 2D, 모두 수작업
      - 디즈니 중심으로 제작, 일본식 2D 제작
      - 오늘날까지 영향 미침
    - 디지털 애니메이션
      - 전통적 기법을 계승해 컴퓨터 그래픽 등 새로운 기술 활용
      - 제작 과정 일부분 또는 전체에 컴퓨터 활용
      - 미래 핵심 산업으로 영화 이외에도 많이 사용

### 애니메이션 역사
    - 초기
      - 고대는 동굴 벽화에 움직이는 동물 그림
      - 19세기는 잔상효과 이용
    - 현대의 애니메이션
      - 플립북: 책 빠르게 넘겨서 움직이는 것처럼 보임
      - 최초의 영화 상영: 1895년 콤마 단위 필름 애니메이션 등장
      - 최초의 영화 상영: 1908년 종이 위에 그림 그려서 연속 촬영해 현대 애니메이션 시작 알림
    - 애니메이션 기법 발전
      - 1914년 공룡 거티는 캐릭터 애니메이션 시초
      - 스톱모션 애니메이션 등장
      - 셀 애니메이션 기법은 제작 용이해 보편적 기법
    - 전통적 애니메이션 시초
      - 필름 제작 애니메이션이 시초
      - 전통 애니 제작 기법은 오늘날 컴퓨터 애니메이션 발전을 위한 기반
    - 전통적인 애니메이션 구분
      - 그림: 종이, 투명한 셀룰로이드 필름위에 손으로 그림 그리고 연속적으로 촬영
      - 모델: 종이, 나무 등 여러 재료를 사용해 인형 물체 등과 같은 모형 제작
      - 제작한 모형을 조금씩 움직이면서 연속적으로 촬영 방식

### 그림 애니메이션
    - 종이 애니메이션 
      - 최초 형태, 잔상효과를 이용한 초기 영사기 원리 이용, 사물 움직임을 한장씩 연속적으로 그려서 연속 촬영
      - 특징: 작가 의도에 따라 여러가지 회화적인 표현 가능, 많은 인력과  작업량이 요구, 제작 기간이 오래 걸리므로 장편은 별로임
    - 셀 애니메이션
      - 오랜 시간이 걸리는 종이 애니메이션 단점 보완
      - 가장 대중적 일반적인 기법, 현재도 많이 사용
      - 투명 셀룰로이드 필름 사용, 애니메이션을 제작
      - 배경은 종이에 그리고 캐릭터는 투명한 필름
      - 배경에 투명한 셀을 올리고 연속적인 그림을 한 프레임씩 끊어서 촬영
      - 키프레임과 인비트윈
        - 숙련된 애니메이터는 키프레임에 해당하는 중요한 장면만 직접 그림
        - 비교적 숙련도가 낮은 애니메이터를 이용해 키프레임 사이에 들어가는 여러장의 중간 프레임을 추정해 그림
        - 인비트윈은 키프레임 사이에 들어가는 프레임
      - 특징
        - 이전의 종이 애니메이션 방식보다 제직 기간 단축, 경비 감소
        - 많은 사람 분업 가능
        - 전경 배경 분리 가능해 상업적 애니메이션에서 가장 적절한 방식
      - 오늘날 셀 애니메이션
        - 90년대 후반까지 아날로그 형식, 문제점 많았음
        - 동화를 제작한 후 셀 복사하는 건 다 수작업
        - 최근은 디지털로 전환되어 시간과 인력이 크게 줄어 듦
      - 유리, 물감 애니메이션
      - 모래 애니메이션
      - 필름 스크래치


### 모델 애니메이션
    - 점토, 종이 등 다양한 재료 사용해 모형을 만들어 제작
    - 스톱모션 애니메이션 기법 사용해 모형을 조금씩 이동해 동작에 변화를 주면서 한 프레임씩 촬영
    - 특징
      - 관절을 움직이는 3차원 모형을 제작해 2차원적인 그림 애니메이션과 같은 융통성 떨어짐
      - 속도감 떨어짐, 분업 작업 불가능해 대량 제작이 어려움
    - 그림 vs 모델
      - 모델링: 촬영 대상을 입체적으로 제작 기술
      - 배치: 모형을 설치된 배경 공간에 적절히 배치
      - 조명: 미니어처 세트를 촬영하기 위한 조명 기술
      - 스톱모션: 한 프레임씩 촬영 기술
    - 종류
      - 인형 애니메이션
        - 인형 기술 + 영화 기술 결합 기법
        - 초기는 나무 인형 -> 철사, 금속, 플라스틱 등
      - 점토 애니메이션
        - 점토 특성을 이용
        - 점토로 모든 캐릭터 물품 제작, 동작을 연속적으로 바꿔서 촬영
        - 플래스티신 같은 점토 최근 이용
      - 컷아웃 애니메이션 
        - 종이로 오려서 한 두 프레임씩 촬영
        - 작은 그림으로도 가능, 하나의 캐릭터가 끝까지 사용되어 표현 세밀하게 가능, 변형 움직임 처리 어렵 
      - 실루엣 애니메이션
        - 유리판 위에 종이 올리고 스톱모션으로 촬영
        - 흑백의 강한 대조를 띠는 특징, 최근 검정 외에 컬러 파라핀 소재 사용
        - 조명 색 변경, 렌즈, 기름층을 이용해 부가적 효과 표현 가능
      - 픽실레이션 애니메이션
        - 인형 애니에 사람을 대상으로 촬영
        - 연기자는 인형, 촬영시에 스톱 모션이 반복되는 동작
        - 재미있는 영상을 표현할때 사용, 픽셀레이션으로만 연출 장면을 요즘은 제작

### 애니메이션 제작 방식
    - 풀 애니메이션
      - 자연스럽고 유연한 동작을 표현하기 위해 초당 그림의 개수를 충분히 그려서 제작
      - 많은 인원, 경비, 기간이 필요
      - 디즈니에서 많이 씀
    - 리미티드 애니메이션
      - 일본에서 제작
      - 시간, 비용 절감을 위해 프레임에서 중간의 그림을 생략
      - 생략과 과장을 통해 독창적 영상 창작
      - 그림의 개수를 줄여 움직임을 표현하기 때문에 캐릭터의 행동이 투박, 딱딱
      - 심슨

### 동작 표현의 개요 
    - 전통 애니 한계
      - 시각 재료만으로 스토리 진행에 따른 움직임 표정 변화 어렵
      - 한계를 극복하기 위해 다양한 동작 표현 기법들을 사용
      - 애니 움직임 표현: 현실 운동 원리 법칙 기본 활용
      - 움직임 표현 요소
        - 낙하 물체의 물리 법칙 등으로 움직임을 제어 조절
        - 현실적인 운동 기본 구성 요소들은 반복, 변형, 과장, 생략
    - 반복
      - 애니에서 가장 많이 사용
      - 여러 프레임을 한 프레임의 루프로 구성
      - 눈 깜박, 걷는 동작은 반복 효과 사용
    - 변형
      - 개체가 찌그러지거나 늘어나는 것과 같이 무게, 유연, 운동, 탄성 표현
      - 변형되어도 물리학 기본법칙에 따라 부피 질량은 안 변함
    - 과장 생략
      - 특정 부분을 원래보다 과하게 표현하거나 생략시키는 기법
      - 과장과 생략 기법을 사용하면 강한 인상 남김
    - 애니메이션 3동작
      - 준비
        - 의도하는 행동 실행 전에 나타나는 움직임
        - 본 동작의 연결과 이어지는 동작 예상
        - 쉽게 인식되지 않기 때문에 연속동작으로 표현하기 위해 필요
      - 본 동작
        - 캐릭터가 의도하는 움직임 의미
        - 전달하려는 의도 표현, 관객들도 이 부분에 집중
        - 의도하는 캐릭터의 표정 감정 움직임의 형태 속도 등 다양한 정보
      - 마무리 동작
        - 움직임이 끝나는 지점에서 반응 여운 등 드러나는 후속
        - 본 동작 이후 자연스럽게 움직임을 종결해 여운 표현 방법

### 캐릭 동작 표현을 위한 로토스코핑
    - 개념
      - 체계적인 캐릭터 윤곽선의 움직임에 따라 배우 연기에 만화 캐릭터의 움직임을 넣어 생동감 구현
      - 연속적인 실사 동작의 이미지를 종이, 셀 위에 투사 프레임 단위로 윤곽선을 복사
      - 실사 이미지 윤곽선을 바탕으로 애니 표현해 원본과 합성
    - 특징
      - 캐릭터가 움직이는 모든 영상을 한 프레임씩 세심하게 그려야 하기에 노동집약적 작업 요구
      - 휴먼, 만화 캐릭터 사이에 자연스러운 상호작용 중요 

### 디지털 애니메이션
    - 컴퓨터 애니메이션을 3D 애니메이션이라 함
    - 디지털 애니메이션의 작업 과정
      - 원화 이미지와 모델들을 스캔해 디지털 데이터로 컴퓨터 저장
      - 데이터의 일부 변수를 바꿔가며 프레임 단위로 애니메이션을 제작해 화면에 연속적 움직임
      - 디지타이저를 이용해 컴푸터에 직접 그리고 이후 단계
    - 발전
      - 셀 애니메이션의 한계를 극복하기 위해 80년대 중반부터 활발
      - 셀 애니메이션 전통 기법을 계승해 컴그 기술과 다양한 기술 요소 제작
    - 표현
      - 표현 방식에 따라 2D 애니메이션과 3D 애니메이션으로 분류
      - 2D는 컴퓨터 2차원 평면으로 표현, 3D는 3차원의 공간
      - 2차원 컴퓨터 그래픽은 낮은 대역폭과 빠른 실시간 렌더링 가능 3D 기술 기반으로 제작
    - 혁신
      - 애니메이션 프로그램은 제작뿐만 아니라 사물 현상을 관찰하기 위한 수단 사용
      - 기술들은 소프트웨어에 탑재되어 일체화
    - 과거 기법의 재발견
      - 최근에 기술적 발전이 아닌 역주행하는 현상도 나타남
      - 기술 도움으로 전통적인 응용 예술에서 축적된 다양한 기법과 기능 적용

### 디지털 애니메이션과 셀 애니메이션 비교 분석
    - 프레임 개수의 변경 여부
      - 셀: 초당 24프레임 제작, 프레임 추가 삭제 불가능한 고정 프레임
      - 디지털: 후반 출력 과정인 렌더링 단계에 필요한 프레임을 미디어 플랫폼 특성에 맞게 조절 가능
    - 색상의 표현 방식
      - 셀: CMYK 기본 색상 표현
      - 디지털: RGB 빛을 이용해 색상 표현
    - 결과물 선명도 차이
      - 셀: 필름, 스캔 절차를 통해 결과물 제작 후 현상을 하는데 작업물의 해상도 손상
      - 디지털: 디지털로 과정이 모두 진행, 현상 과정 없이 렌더링으로 최종 결과물을 완성, 높은 해상도
    - 선화, 동화 작업의 자동화
      - 다양한 애니메이션 저작도구에 모션 편집기 기능 탑재
      - 원화 생성, 부위별 레이어화, 기본 움직임 표현, 변형 등 다양한 종류 기능 적용해 애니 제작
      - 프레임마다 변화되는 셀의 복잡한 선화, 동화 작업 자동화
    - 이펙트 및 다양한 적용
      - 셀: 셀 겹침의 한계점으로 배경과 이펙트가 단순
      - 디지털: 여러 기법을 이용해 다양한 효과를 적용

### 디지털 애니메이션 발전 과정
    - 90년대 이후 빠르고 저렴하게 그래픽 이미지를 처리하는 환경 구축
    - 3D 모델링, 렌더링, 애니, 동화 등 손쉽게 처리하는 소프트웨어
    - 기술적인 차원에서 비약적으로 발전, 광선 추적법, 실시간 애니메이션 등으로 완성도 높은 애니 제작
    - 다양한 시각화, 모의 비행 장치에서도 사용

### 컴퓨터 이미지 작업 발전 과정
    - 벨 연구소에서 통신 위성 움직임 디자인하기 위해 고안
    - 추상적인 영화 제작을 위해 사용
    - 영상 품질이 향상되면서 실제로 현존하지 않는 세계를 매우 사실적으로 보이게 만들 정도로 발전

### 디지털 애니메이션 특징
    - 3D 디지털 애니는 컴퓨터 사용 -> 3차원 가상 공간에 제작
    - 소프트웨어 발달로 더 정교, 실사에 가깝
    - 카메라 움직임, 각도 등 변화시키면 역동적인 장면 만들어짐

### 캐릭터 모델링
    - 하나의 캐릭터 모델링 데이터 재활용해서 다양하게 변형 또다른 모델 가능
    - 3차원 만든 캐릭터는 외형적 특징 강조
    - 3D 애니는 컴퓨터로 계산해 캐릭터 움직임
    - 사람 캐릭터를 사실적 표현은 어려움

### 제작 기술
    - 전통적인 수작업 기법들을 소프트웨어적인 프로그램으로 전환
    - 모든 과정을 컴퓨터로 진행해도 개인의 독창적인 아이디어를 그림으로 직접 그려야 함

### 제작 
    - 기획 : 전통적인 방식으로 작품 기획, 시나리오, 캐릭, 도구 디자인
    - 프리프록덕션: 2차원 배경 및 캐릭 모델과 입힐 매핑과 택스처 미리 제작
    - 프로덕션: 3차원 사물 직접 그리는 모델링 작업, 레이아웃, 원화, 피사체 움직임 부여 동화 그림
    - 디지털화: 원화, 동화를 컴퓨터에 입력해 트레이스 과정과 채색 과정 거쳐 배경 그림, 
      - 렌더링: 모든 데이터를 계산 실제 이미지 구현
    - 영상 연출: 채식이 완료 그림에 특수효과 등을 추가해 촬영 및 편집
    - 편집, 녹음: 배경음악, 성우 더빙 결합해 최종 완성
