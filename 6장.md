### 컴퓨터 그래픽 기술의 진화
    - 컴퓨터 그래픽 기술
      - 컴퓨터를 사용하여 제작된 이미지와 이미지를 구현하기 위한 기술 통합
      - 사용자 컴퓨터 상호작용을 향상
      - HMD 개념, 게임, 애니메이션 등 광범위하게 활용

### 컴퓨터 그래픽 기술 활용
    - 3D 시물레이션
      - 제품이 디자인대로 작동하는지 예측
      - 비행기 설계 -> 제조업 전반에 확산 후 보편화
      - 초기에는 3D 설계 -> 디지털 실물 모형 제작 수준 발전
      - 2000년대 이후로 제품의 설계, 생산, 관리에 이르기까지 전 과정 경험 개념으로 진화
    - 영화 분야의 디지털 기술
      - 터미네이터 2: 인간 캐릭터를 그래픽으로 변형시키는 몰핑 기법
      - 토이스토리: 최초의 100% 3D CG 장편 애니메이션
      - 스타워즈 1: 세계 최초 디지털 영화, 전체 필름 디지털화
      - 아바타: 영화 제작과 상영 모든 과정을 컴퓨터 기술 처리

### 2D 컴퓨터 그래픽 개요
    - 2차원 평면에 X, Y 좌표로만 구현
    - 2D 애니메이션, 일러스트 문자 디자인 등
    - 래스터 그래픽 방식 사용 -> 벡터 그래픽 방식 발전
    - 사실적 그림체인 만화, 애니메이션 분야에서 2D 기술은 더 중요

### 3D 컴퓨터 그래픽 개요
    - 2차원 이미지를 3차원 공간에 존재하는 형태로 인식
    - 모델의 기하학적 데이터를 3차원우로 표현 후 2차원에 나타냄
    - 제작과정
      - 1. 대상을 3차원으로 형상화하는 모델링
      - 2. 장면에 조명을 비추어 음영을 표현하는 셰이딩, 질감, 기타 효과를 넣음
      - 3. 명령으로 연산처리를 해 최종적인 그래픽 이미지를 출력하는 렌더링 과정

### 조명의 개요
    - 조명
      - 전체적인 느낌과 분위기를 표현하기 위해 인위적으로 사용하는 광선
      - 영화, TV, 컴퓨터 그래픽, 사진 등과 같은 시각 매체에서 사용
    - 조명 구성
      - 주광선
        - 앞쪽에서 피사체를 비추는 기본 광선
        - 피사체에 그림자를 만듬, 주광선의 세기, 색깔, 각도에 따라 피사체의 형태, 표면 질감 등 물체의 전반적인 모습이 결정
      - 보조 광선
        - 물체를 측면에서 비추는 광선
        - 주 광선의 그림자 부분을 비추어 제거, 명암의 대조를 감소시켜 물체를 부드럽게 보이게 함
        - 주광선보다는 덜 밝고 부드러운 조명 이용
      - 후면 광선
        - 주광선의 반대편에 비춰지는 역광선 의미
        - 화면의 평면화를 방지해 피사체의 입체감을 더함
        - 역광은 배경과 피사체를 분리해 피사체의 형태, 배경, 피사체 등 과의 관계를 명확하게 함
      - 3차원 컴퓨터 그래픽
        - 조명에 의한 음영을 표현하는 셰이딩 처리 과정을 거쳐 입체적 이미지
        - 초명의 질, 색상, 강도, 방향성에 따라 품질 결정
        - 최근 실시간 구현 기술 개발

### 조명의 정류
    - 부분 광원
      - 장면을 강조하거나 효과를 극대화하기 위해 일부분 비추는 조명
      - 활용 범위 다양
    - 방향성 광원
      - 태양광처럼 일정한 방향으로 동일한 광선량을 투사하는 조명
      - 전체에 동일한 각도로 조사, 그림자를 만들고 뚜렷한 입체감, 질감 표현
    - 전방향성 광원(포인트 라이트)
      - 백열전구와 같이 한 지점으로부터 모든 방향으로 빛을 비춤
      - 광범위하게 쓰임
    - 스팟 광원
      - 무대 조명으로 많이 사용
      - 방향을 제시, 거리감 주고 조명각 설정 가능, 특정 부위를 집중적으로 비추는 부분 광원
    - 분위기 광원(환경광)
      - 주변에 여러 종류의 광원에 의한 빛의 반사와 산란 등으로 발생한 광원
      - 방향과 거리가 없는 가상의 조명
      - 이미지 전체에 골고루 영향, 모든 대상을 일정하게 비추고 그림자 없음

### 3d 모형 제작 과정
    - 모델링: 물체를 포현
      - 3D 가상공간에 수학적 모델 이용해 3차원 모형 만듦
      - 폴리곤 방식: 모형의 모든 면을 삼각, 다각형으로 표현
      - 스플라인 곡선 방식: 곡선으로 일정한 형식의 뼈대를 만든 다음 곡면을 표현해 형상화
    - 셰이딩: 음영 처리
    - 매핑: 실제 물체와 비슷한 질감 표현
    - 렌더링: 연산 처리로 최종적인 화면을 만드는 과정
      - 모델링을 통해 3차원 모형을 재질, 조명 설정, 실제와 비슷한 질감으로 표현되는 최종 결과물

### 3D 모델링의 종류 특징
    - 와이어 프레임 모델링
      - 가장 오래된 방식, 골격만 모델링
      - 메시 패턴의 그리드 모양의 선 표현 모델
      - 데이터 용량이 적어 모델링에 걸리는 시간이 빠름
      - 보이지 않아야 할 모델의 뒷부분이 보이는 단점
    - 폴리곤 모델링
      - 다각형 면의 집합으로 물체를 표현하는 방식
      - 물체를 모델링할 때 모형의 모든 표면을 삼각형 또는 다각형으로 표현
      - 수많은 폴리곤이 모여 3차원 모형
      - 폴리곤이 많으면 세밀하게 표현, 완성도 향상, 많은 시간 소요
      - 제작이 쉽다, 시스템 자원을 적게 사용해 빠른 작업 속도
    - 특징
      - 모델의 표면을 3차원으로 표현
      - 과정, 완성 후에도 표면을 추가 변형 쉬움
      - 공간 좌표로 논리 연산 가능
      - 유선의 곡면은 폴리곤을 분할 다수의 작은 폴리곤 처리
      - 곡면 확대시 평면
      - 점, 선분, 메시, 경계로 구성
    - 서피스 모델링
      - 와이어 프레림에 천을 씌어놓은 형태로 내부는 빈 공간, 표면과 안쪽면이 존재
      - 폴리곤을 기본 단위로 표면 모델링
      - 데이터 양은 와이어 보다 많지만 솔리드 모델보다 적음, 고속 렌더링 가능
      - 방식에 따라 폴리곤 메시와 스플라인 곡선 2가지 분류
    - 폴리곤 메시
      - 면을 기본으로 모델링
      - 간단해서 만들기 쉽지만 곡선 어렵
    - 스플라인 곡선
      - 형태를 조절하는 제어점을 가진 곡선, 변형 가능 선
      - 스플라인으로 일정한 혁싱에 따라 뼈대 만들고 곡면 표현해 형상화
    - 넙스 모델링
      - 일정한 점들을 스플라인 공식을 적용 후 곡선 변환 후 기본으로 곡면을 표현
      - 표면을 확대해도 곡면 유지, 3D객체를 가장 섬세, 정교하게 모델링
      - 특성상 직선 표현 불가능, 각진 표현 불가능
      - 시스템 자원 많이 사용해 느림
      - 특징
        - 조절점을 계산해 연결하기 때문에 처리 시간 많이 걸림
        - 논리 연산, 테두리를 손질하는 트림 연산 가능
        - 넙스 모델은 포릴곤 모델로 전환이 가능해 게임 개발에는 폴리곤 모델로 전환해 사용
    - 솔리드 모델링
      - 입체 내부까지 채워 볼륨있는 객체 표현
      - 기본체들을 합치고 깎으면서 만듦
      - 모델 내부가 채워진 덩어리 형태로 입체적인 모양 구현
      - 3D 객체의 내부 질량, 부피, 강도 등 같은 데아터 계산 가능

### 3D 렌더링 개요
    - 3D 렌더링
      - 2차원 이미지 모델링한 피사체에 광원, 위치, 색상 등 외부의 정보를 고려해 입체감 있는 3차원 이미지 표현
    - 렌더링 알고리즘
      - 최종적으로 완성된 그래픽 이미지의 품질 결정
      - 많은 명령을 수행, 많은 픽셀 값들을 계산해 처리시간 김
      - 렌더링 과정은 사용되는 알고리즘에 따라 여러가지 방식 존재

### 투영과 은면 제거
    - 투영법
      - 3차원 객체 형태, 크기 등을 일정한 법칙으로 2차원 평면 공간에 정확하게 표현
    - 평행 투영법
      - 투영의 중심을 투영면에서 무한한 원점으로 생각했을때 투영 방법
      - 투영선은 모두 평행을 이루고 있음
      - 주로 기계의 설꼐 도면에 사용
    - 투시 투영볍(원근 투영법)
      - 어느 시점에서 비평행선의 투영선에 의하여 투영된 것을 나타냄
      - 사진 기술, 사람의 시각과 유사한 효과를 위해 투영점에서 떨어진 거리에 따라 물체의 크기를 조절
      - 주로 건축 도면에서 사용
      - 물체의 정확한 모양, 깊이, 각도로 표현해야하는 응용에는 부적합
      - 3D 객체를 좀 더 자연스럽게 표현 가능
    - 은면 제거
      - 은면: 표현된 객체의 보이지 않는 표면은 보는 위치에 따라 보이는 표면과 보이지 않는 표면
      - 3D 그래픽에서는 모든 표면에 대해 은면 제거 과정을 수행해 어떤 표면이 화면에 나타날것인가를 검출
      - 3차원의 물체를 2차원 화면에 사실적 표현
      - Z 정렬, Z 버퍼, 주사선, 광선 투사법 등 있음
      - 은면 제거 기법에 따라 렌더링 기법 종류와 특성 결정

### 셰이딩
    - 개요
      - 렌더링 과정에서 빛의 거리와 각도에 따라 모형 표면의 조도 변화
      - 물체 표면에 색상을 입혀 입체감, 질감 표현
      - 광원으로부터, 거리, 각도, 색상, 명암 등을 계산해 음영 표현
      - 조명의각도에 따라 모든 표면의 밝기가 다르게 표현
      - 렌더링 과정에서 가장 중요
    - 와이어 프레임 셰이딩
      - 수많은 선을 결합해 입체적으로 만든 뼈대 형태의 3D 모형에 음영 표현
      - 와이어 프레임 구조의 3D 객체가 현실감 떨어짐, 보이지 않는 선 제거하는 은선 또는 은면 제거 작업 수행
      - 면에 적절한 색상, 음영 표현, 구조가 간단해 표현하는 시간 적게 걸림
      - 셰이딩 방식중 가장 간단, 처리속도 빠름
    - 플랫 셰이딩
      - 3D 모형을 구성하는 폴리곤들의 표면을 단색으로 채우는 방법
      - 표면 모서리 경계 부분에 개별적인 색상을 지정하거나 급격한 명암차
      - 와이어 프레임 셰이딩 보다 발전된 방식, 폴리곤 표면마다 단색 지정 -> 렌더링 빠름
      - 단조롭고 질감이 거칠어 부드럽지 못함
    - 스무드 셰이딩
      - 부드럽고 사실적 표현이 가능한 고급 셰이딩 기법
      - 고라우드 셰이딩, 펑 셰이딩으로 진화
    - 고라우드 셰이딩
      - 3D 객체의 표면에 연속적인 음영을 나타내기 위해 보간법 사용
      - 모든 삼각형 폴리곤에 대해 꼭짓점의 조명을 계산, 삼각형 표면에 연속적인 조명 표현
      - 플랫 셰이딩보다 부드러운 명암을 표현 가능, 펑 셰이딩 보다 처리 속도 음영 품질 우수
      - 빠르고 단순하지만 폴리곤 모서리 경계선 모양이 나타나는 단점
    - 펑 셰이딩
      - 고라우드 셰이딩에서 부자연스러운 매끄러움 문제 해결
      - 3D 모형의 표면을 구성하는 픽셀들에 대해 조명 색상을 개별적으로 계산해 실제와 가장 유사하게 표현
      - 표면에 나타나는 모서들을 감추기 때문에 곡선 표면 물체에 적합
      - 가장 수준 높은 렌더링 가능, 각 픽셀에 조명 색상 계산하기 때문에 시간 많이 소모
      - 가장 많이 쓰이는 셰이딩 방법, 플라스틱 질감 표현하기에 적합

### 3D 매핑
    - 2차원 이미지를 굴곡이 있는 3차원 입체적인 표면으로 옮겨 표현
    - 물체가 가지고 있는 고유의 재질, 표면의 색상, 패턴 등을 표현
    - 텍스처 매핑
      - 3D 객체 표면에 질감 데이터 입힘
      - 2차원 이미지가 3차원 객체의 표면에 칠해진 것과 같이 표현
      - 패턴 또는 무늬 사용, 폴리곤 같은 단편의 색 결정
      - 빛을 사용해 물체 표면 밝기, 빛의 반사, 굴절 등을 고려해 질감 표현
    - 범프 매핑
      - 3D 객체의 표면이 불규칙, 굴곡이 있으면 물체 표면을 구성하는 픽셀마다 수치 변경해 엠보싱 효과
      - 조명으로 각 픽셀의 밝기와 음영 조절, 텍스처 매핑과 병행해 이미지 더 현실적
      - 실제로 존재하지 않는 돌출 부위를 존재하는것처럼 표현
      - 많은 데이터와 빠른 처리 요구
    - 디스플레이먼트 매핑
      - 3D 객체의 표면을 뒤틀어 변형하는 기법
      - 픽셀들을 공간적으로 이동 실제로 굴곡진 효과
      - 굴곡의 형태를 도형에 직접 적용, 표면에 도형 자체를 변화시키기 때문에 객체의 윤곽선이 울퉁불퉁
      - 표면을 강하게 변화시키기 때문에 범프 매핑보다 사실적
      - 렌더링 시간이 범프 매핑에 비해 많이 소요
      - 실시간 처리에는 사용하지 않고 광선 투사 이미지 사용
    - 오패시티 매핑
      - 3D 객체 표면을 투명, 불투명 부분으로 구분해 표현
      - 명도 단계에 따라 객체 투명도 조절
      - 흰색 부분은 투명, 검은색 부분은 불투명
      - 투명한 부분은 하이라이트를 만들어 유리효과도 냄
    - 리플렉션 매핑
      - 객체가 조명을 받아 반사체 주위의 환경이 비쳐 보이는 것을 표현
      - 금속, 유리 등 투명체를 통해 보는 경우에 발생하는 굴절 현상 표현
      - 주변 물체가 반사체에 나타나는 굴절 현상의 정확성이 광선 추적법보다 떨어짐
      - 반사 이미지를 미리 렌더링해 계산량이 적어 렌더링 시간 감소
    - 라이트 매핑
      - 조명을 사용해 3D 객체의 표면을 표현하는 기법
      - 이미지 표면에 라이트 맵을 적용해 새로운 이미지를 만듬
      - 일반적으로 미리 계산되어 정적인 모형에 사용
      - 아주 큰 평면의 표면, 도시, 실내 환경 표현에 적합
      - 실시간 조명 처리와 픽셀당 조명 처리 방식 대체

### VFX 기술
    - CG를 통해 연출
    - 영상 제작 기법 중 컴퓨터 그래픽을 기반으로 표현하는 모든 디지털 기법 의미
    - ICT 환경이 발전하면서 시각 효과 관련 기술 많은 발전
    - 유체의 흐름을 재현하기 위해서 나비에 스톡스 방정식이 기본
    - 모아나에는 APIC 기법이 적용
      - APIC: 기존과 같은 조건에서도 움직임이 오래 지속, 안정적이면서 자연스러운 CG
