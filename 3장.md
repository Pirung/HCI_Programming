### 멀티미디어 시스템의 개념
    - 멀티미디어 시스템
      - 멀티디미어 시스템: 다양한 작업을 처리하는 컴퓨터 또는 디지털 기기
      - 컴퓨터, 스마트 폰, 태플릿 pc등

### 멀티미디어 시스템의 구성 요소
    - 하드웨어
      - 각종 디바이스에서 입력되는 아날로그 신호를 0,1 로 변환
    - 소프트웨어
      - 시스템 소프트웨어
        - 운영체제: 사용자가 쉽게 사용할 수 있도록 인터페이스 제공
        - 데이터베이스: 데이터를 검색, 갱신, 삽입, 삭제
        - I/O 장치 드라이버: 다양한 종류의 하드웨어를 제어
        - 통신 프로토콜: 컴퓨터끼리 통신할 수 있도록 정의한 양식 또는 규칙
      - 응용 소프트웨어
        - 사용자가 원하는 목적에 따라 시스템에게 작업을 지시하기 위한 프로그램
      - 프로세서
        - 코어: 프로세서에 내장된 처리 장치의 핵심 부품
      - 그래픽 처리장치
        - 그래픽카드: 데이터를 모니터에 전달해주는 장치
          - CPU가 그래픽카드 비디오 메모리에 데이터를 저장
          - GPU가 저장된 정보를 아날로그 영상신호로 바꾸어서 모니터에 전달

### 기억장치의 계층 구조
    - 기억장치
      - 임시 기억장치, 주기억장치, 보조 기억장치
      - 특성에따라 처리속도 모두 다름
      - 그대로 사용하면 가장 느린 기억장치의 속도에 맞춰서 작업
      - 기억장치를 성능, 용량, 비용등에 따라 계층구조로 배치하면 시스템의 처리능력 향상

      - 주기억장치
        - RAM: 휘발성 메모리

### 저장장치
    - 개념:
      - 컴퓨터의 외부에 정보를 영구적으로 보존하는 비휘발성 기억장치
      - 주기억장치에 비해 저렴, 느림
      - 플로피디스크, 하드디스크 등
    - 하드디스크:
      - 알루미늄판의 표면에 자성 물질을 입힌 딱딱한 원판으로 된 자기디스크 장치
      - 실린더, 트랙, 섹터의 번호를 주소로 사용
      - 대용량 스토리지 시장이 확장에 따라 재조명
      - 클라우드 서비스 확장, CCTV 수요 증가, 대용량 데이터를 위한 저장장치의 필요성
    - 광디스크
      - 레이저를 이용해 정보를 기록, 재생하는 대용량 기억장치
      - 최근 쇠퇴
    - 낸드플래시 메모리
      - 전원이 꺼져도 데이터 보존 비휘발성 메모리
      - 좁은 면적에 고밀도 집적으로 대용량화 가능
      - 가격이 저렴 NOR 플래시에 비해 빠름
      - 데이터를 저장하는 셀의 능력에 따라 진화
    - SSD 메모리
      - 낸드플래시 적용
      - HDD에 비해 빠름
      - 낸드플래시 가격이 떨어지면서 HDD 대체

### 반도체 산업의 미래
    - 미국: 메모리와 시스템 반도체 분야에서 높은 시장 점유율
    - 아시아: 중국은 메모리 반도체에 집중, 일본은 소재 장비 분야에서 최고 경쟁력
    - 국내: DRAM 삼성 SK 마이크론, 공급 과잉 문제 중국때문에

### 시스템 반도체 산업
    - 시스템 반도체 개요
      - 빛, 소리, 영상, 압력, 온도 등 아날로그 신호를 디지털 신호로 바꿔주는 반도체
      - 연산, 제어, 전송 등 여러 기능을 하나로 통합한 칩으로 운영에 필요한 설계 기술이 경쟁력
    - 팹리스: 반도체를 생산하는 팹을 직접운영 안함, 설계와 기술 개발을 전문
    - 파운드리: 팹에서 생산만 담당

### 클라우드와 네트워크 결합 스토리지
    - 클라우드 저장 서비스
      - 인터넷에 존재하는 클라우드 공간에 데이터를 저장
      - 인터넷 연결 데이터 백업, 재생, 최신사양과 스마트 오피스 제공
      - 해킹에 의한 자료 유출 위험 제거
    - 네트워크 결합 스토리지
      - 클라우드 서비스 유사
      - 시스템 구축과 서비스를 사용자가 직접 관리
      - 동일 용량의 클라우드 서비스보다 저렴하고 안전하게 데이터를 활용
      - 개인 사용자, 소규모 기업 구축하는 폐쇄형 클라우드

### 입력장치
    - 필요한 데이터를 컴퓨터로 읽어 들이는 장치
    - 멀티미디어 정보를 디지털 형태로 변환
    - 초기에는 포인터 이동, 터치만
    - 지금은 2개 손가락 인식 가능

### 출력장치
    - 모니터
      - CRT, LCD LED 등
      - LCD의 비율 높음, 차세대 디스플레이 OLED
    - 프린터
      - 컴퓨터에서 처리된 결과 인쇄
      - 프린터의 성능을 나타내는 단위
        - DPI 프린터 해상도, 1인치에 도트가 얼마나 들어가냐
        - CPS 인쇄 속도, 초당 인쇄되는 문자 수
