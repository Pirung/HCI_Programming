# 1장
### 멀티미디어 탄생
-------------
    - 90년대 이후 ICT 환경이 발전 -> 디지털 형태로 변환
    - 누구나 멀티미디어 제작 가능
    - 효과적인 소통 수단, 콘텐츠의 질이 상승
    
    멀티미디어 개념
    - 다수(Multum) + 정보 유형(medium)
    - 디지털화한 복합 구성 매체
    - 디지털 미디어를 2개 이상 결합해 목적을 가진 콘텐츠 생성
    - ICT 환경을 기반으로 시스템과 상호작용해 지식 정보 제공하는 기술

    - 사운드, 비디오: 연속적, 현실 세계에서 수집
    - 애니메이션: 연속적, 컴퓨터에 의한 합성
    - 이미지: 이산적, 현실 세계에서 수집
    - 텍스트, 그래픽: 컴퓨터에의한 합성, 이산적

    멀티미디어 특징
    - 현실과 유사한 환경을 콘텐츠
    - 정보의 비선형적 탐색: 사용자 우선, 낮은 비용, 시간 공간 제약 초월
    - 사용자 간 상호작용
    - 대용량 멀티미디어 데이터 처리와 검색
        실시간 처리, 복제성(무한 저장 복제 변환 가능), 데이터 저장, 빠르게 정보 검색
    - 콘텐츠 유통 방식의 변화
        시공간 초월해 원하는 정보 검색, 가공, 저장, 공유 가능  
    
    표현 미디어 -> 저장 미디어 -> 전송 미디어 
    -> 제시 미디어 -> 지각 미디어

    전통 미디어
    - 정보통신 기술이 도입되기 이전부터 사용된 통신 수단
    - 레거시 미디어: 전통적인 언론, TV, 신문, 잡지, 라디오
    - 인쇄 전파 비디오 뉴미디어 시대로 발전,
    - ICT 기술을 기반으로 한 미디어 시장의 진화와 변화, 현장감, 속도감이 부족해 하락

    뉴미디어
    - 전통 미디어와는 다른 전송 방식과 콘텐츠를 사용하는 제품이나 서비스
    - 전통 미디어 영향력 감소, 인터넷 포털 매체가 각광
    - 전통 미디어를 복합적으로 활용, 단순한 정보 전달뿐 아니라 콘텐츠 생성 기능까지 포함
    - 2010년대 이후 본격화, 미디어의 순환 구조에 전체적인 변화와 기능의 근본적 변화
    - SNS, 상호작용 뉴스

    뉴미디어 특성
    - 정보의 디지털화: 컴퓨터 이용으로 인해 모든 정보 디지털 신호
    - 정보의 통합화: 아날로그 형태의 매체가 비트로 통합
                    정보가 유기적으로 융합되어 사용자에게 전달
    - 정보의 영상화: 정보가 영상으로 표현 다양한 디바이스 화면을 통해 전달
    - 정보의 상호작용성: 쌍방향 전송 가능, 원하는 정보만 선택, 정보 제공자와 사용자간 상호작용을 통해 새로운 정보 생성
    - 정보의 비동기화: 시간 공간 제한 극복
    - HCI 기술과 멀티미디어의 융합: 인간 컴퓨터 상호작용 기술이 멀티미디어 환경에 적용해 인간이 컴퓨터를 더 유용하게 사용

    스마트 미디어
    - 사용자와 상호작용 가능, 시공간 제약 없이 융복합 콘텐츠 제공 매체
    - 쌍방향 미디어, 상호작용성을 통해 타인과 자유롭게 소통 가능
    - 스마트폰 등장 후 시공간 제약 극복
