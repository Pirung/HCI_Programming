# 1장
    멀티미디어 탄생
    - 90년대 이후 ICT 환경이 발전 -> 디지털 형태로 변환
    - 누구나 멀티미디어 제작 가능
    - 효과적인 소통 수단, 콘텐츠의 질이 상승
    
    멀티미디어 개념
    - 다수(Multum) + 정보 유형(medium)
    - 디지털화한 복합 구성 매체
    - 디지털 미디어를 2개 이상 결합해 목적을 가진 콘텐츠 생성
    - ICT 환경을 기반으로 시스템과 상호작용해 지식 정보 제공하는 기술

    - 사운드, 비디오: 연속적, 현실 세계에서 수집
    - 애니메이션: 연속적, 컴퓨터에 의한 합성
    - 이미지: 이산적, 현실 세계에서 수집
    - 텍스트, 그래픽: 컴퓨터에의한 합성, 이산적

    멀티미디어 특징
    - 현실과 유사한 환경을 콘텐츠
    - 정보의 비선형적 탐색: 사용자 우선, 낮은 비용, 시간 공간 제약 초월
    - 사용자 간 상호작용
    - 대용량 멀티미디어 데이터 처리와 검색
        실시간 처리, 복제성(무한 저장 복제 변환 가능), 데이터 저장, 빠르게 정보 검색
    - 콘텐츠 유통 방식의 변화
        시공간 초월해 원하는 정보 검색, 가공, 저장, 공유 가능  
    
    표현 미디어 -> 저장 미디어 -> 전송 미디어 
    -> 제시 미디어 -> 지각 미디어

    전통 미디어
    - 정보통신 기술이 도입되기 이전부터 사용된 통신 수단
    - 레거시 미디어: 전통적인 언론, TV, 신문, 잡지, 라디오
    - 인쇄 전파 비디오 뉴미디어 시대로 발전,
    - ICT 기술을 기반으로 한 미디어 시장의 진화와 변화, 현장감, 속도감이 부족해 하락

    뉴미디어
    - 전통 미디어와는 다른 전송 방식과 콘텐츠를 사용하는 제품이나 서비스
    - 전통 미디어 영향력 감소, 인터넷 포털
